This is how to use vim for begginers.

MOVING THE CURSOR

In order to move the cursor, press h,j,k,l keys as indicated or press the arrow keys on the bottom right of your keyboard.

  • Hint: The h key moves left.
  • The l key moves right.
  • The k key moves up.
  • The j key moves down.

EXITING VIM

  • 1. Press ESC key (to make sure your in Normal mode)
  • 2. Type: :q!
  • This exits vim, DELETING any edits or changes you have made.
  • 3. If you want to SAVE your changes in your file, use: :wq
  • 4. If you know these commands and you are confident then you are all good to exit vim without issues.

GOING INTO VIM THROUGH TERMINAL

  • 1. In terminal start with typing in: cd desktop
  • This sends you into your files in desktop.
  • 2. Next type in: ls
  • This lists all your files so that you can choose the one you need to edit.
  • 3. Once you choose your file, continue with typing in: cd file(for example)
  • This lists all the smaller files inside the file.
  • 4.If you want to create a new file type in: vim newfile(for example)
  • !Now you are in vim!

INSERT MODE

  • This step is really easy, in order to get in Insert mode type: i
  • But if you want to EXIT Insert mode type: ESC
  • This brings you in Normal mode.
  • If you understand this, then you are all set!
